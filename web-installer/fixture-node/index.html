SIGHTLINE USB Installer

Flash SIGHTLINE Fixture Node firmware directly from your browser over USB.

This page was opened with file://. The installer must be served over http://localhost or HTTPS so the manifest can load.

Install Steps

  1. Connect your ESP32 over USB.
  2. Click Connect / Install above.
  3. Select the ESP32 serial port in the browser prompt.
  4. Wait for flashing to complete.
  5. Open serial monitor at 115200 for first-boot logs.

After Flashing

If no valid network config is saved, the device starts in setup AP mode. Connect to the setup AP and configure Ethernet or Wi-Fi Station.
